# Homebrew Tap for d-spec

This is the Homebrew tap for [d-spec](https://github.com/dwrekofc/d-spec), a planning/ideation workflow template for AI-assisted development.

## Installation

```bash
brew tap dwrekofc/d-spec
brew install d-spec
```

## Usage

```bash
# Initialize d-spec in your project
cd your-project
d-spec init

# Show help
d-spec help

# Show version
d-spec version
```

## Requirements

d-spec requires [beads](https://github.com/steveyegge/beads) for issue tracking:

```bash
brew tap steveyegge/beads
brew install bd
```
